<p>When passing a value for a gradient, you must specify the position of all colors. Otherwise the component will throw an exception.
For example:</p>

<pre><code><span class="hljs-function"><span class="hljs-title">linear-gradient</span><span class="hljs-params">(<span>180deg</span>, <span>#000000</span>,<span>#ff0000</span>)</span></span>

</code></pre>
